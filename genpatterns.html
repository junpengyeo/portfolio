Pattern Generation

This project is designed for Arts@Metta to enhance the corporate value and commercialbility of the organisation's product offerings. A corporate logo is used as a starting point for the design, first being dissected and then distributed randomly using an algorithm, generating an countless pattern possibilties.

Client

Arts @ Metta

Created

2020

Role

Designer

Logo Extraction

Three or more different shapes are first extracted from the corporate logo which will be used by the algorithm as elements for the pattern design.

With the extracted elements, the algorithm will apply multiple random parameters such as scale, rotation, quantities and spacing to them before being distributed as a pattern as shown.

Distributed Patterns

Product Examples

These printed patterns can be integrated into Metta's exisiting product lines to increase the corporate attractiveness of their products. For instance, the DBS logo can be imprinted on to their pottery vases to create a uniquely DBS branded vase or a picture from for Allianz.
